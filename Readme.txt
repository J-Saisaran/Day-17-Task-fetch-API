# Day 17 Task: Fetch API - Rest Countries and OpenWeatherMap Integration

This project demonstrates the integration of the Rest Countries API and OpenWeatherMap API using the `fetch()` API in JavaScript. 
It fetches data from the Rest Countries API to display country details, including the capital, region, country code, latitude, and longitude. Additionally, it provides weather information for the selected country's capital using the OpenWeatherMap API.


## Features

- Fetches and displays a list of countries from the Rest Countries API.
- Displays country details in Bootstrap cards, including:
  - Country name
  - Country flag
  - Capital city
  - Region
  - Country code
  - Latitude and Longitude
- Fetches and displays the current weather for the selected country's capital using the OpenWeatherMap API.
- Responsive design using Bootstrap.
